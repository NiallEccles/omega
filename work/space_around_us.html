Space Around Us

Space Around Us is an educational website that aims to teach children about some of the planets in our solar system.

This website was created as my college final year project. I had always wanted to create an educational space website and this was the perfect time to do it. Initially the website was supposed to feature all of the planets, but was revised to only include 4 planets, Mars, Jupiter, Saturn and Uranus.

The website was created using HTML, CSS and JavaScript. I also used jQuery, Sweet Alert and fullPage.js. Sweet Alert allowed me to create modal boxes that provide the user with more information and facts. fullPage.js provides the slides for each page.

I learned a lot about how I work as a developer and also how I design. SVG's were used thoroughout the site as a way to have full screen scenes. I could then apply more complex animations to the SVG's.
